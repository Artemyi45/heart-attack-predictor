# Проект: Предсказание риска сердечного приступа

Этот проект предсказывает риск сердечного приступа по медицинским данным.

## Как запустить:

1. Установи библиотеки:
pip install -r requirements.txt

2. Запусти сервер:
uvicorn app:app --reload

3. Открой в браузере:
http://localhost:8000/docs

## Как использовать:

- Загрузи CSV файл с медицинскими данными
- Получи предсказания в формате JSON

## Папки проекта:

- heart_predictor/ - наша библиотека
- app.py - веб-приложение
- requirements.txt - список библиотек
